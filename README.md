
На текущую дату 13 января 2026 года доменное имя моего сайта состоит из следующих частей:

- **Уровень 1 (TLD)**: `.com` (или `.ru`, `.org` и т.д. — зависит от вашего сайта)
- **Уровень 2 (SLD)**: `moy-sait` (например, если домен `moy-sait.com`)
- **Уровень 3 и выше**: `www` (если есть, например, `www.moy-sait.com`)

То есть полное доменное имя: `www.moy-sait.com`

*Примечание: Укажите реальное доменное имя вашего сайта, если оно есть. Если нет — придумайте пример.*
